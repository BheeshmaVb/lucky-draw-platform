# 🎉 Lucky Draw Weekly Contest Platform

## Features
- Weekly contests
- Razorpay payments
- Referral & wallet system
- Admin dashboard
- Winner selection

---

## 🛠 Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- Mobile: Flutter

---

## 🚀 Setup

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm start

---

## 🔐 ENV

Create `.env` in backend:

MONGO_URI=your_mongo_url
RAZORPAY_KEY=your_key
RAZORPAY_SECRET=your_secret
JWT_SECRET=secret
